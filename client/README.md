# Client — Do You Need to Install Anything?

**Short answer: No.** For the core setup, your friends can join with plain vanilla Minecraft.

## Core setup (no install required)

All the performance mods (Lithium, FerriteCore, Krypton, ModernFix, ScalableLux, Noisium) run **on the server only**. Players connect with a standard, unmodified **Minecraft Java Edition 1.21.1** launcher. There is nothing to download or configure — all they need is the playit address you share with them.

See [PLAYER-GUIDE.md](PLAYER-GUIDE.md) for step-by-step joining instructions.

## Optional: content mods

If you decide to add visual or item mods to your server later, players will need a matching modpack. That is distributed as a `.mrpack` file and installed via Prism Launcher. This is purely optional and is covered in the adaptation guide:

→ [`../method/06-uyarlama-rehberi.md`](../method/06-uyarlama-rehberi.md)
