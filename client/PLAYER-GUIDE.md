# Player Guide — Joining the Server

Welcome! This takes about 3 minutes. Pick your path below.

---

## Path A — Vanilla (no mods)

This is the normal case. You do **not** need to install any mods.

1. **Launch Minecraft Java Edition 1.21.1.**
   Make sure your launcher is set to version `1.21.1`. Any standard launcher works (official Minecraft Launcher, Prism, etc.).

2. **Go to Multiplayer → Add Server.**

3. **Paste the playit address** your friend (the server owner) shared with you into the *Server Address* field.

4. **Click Done, then Join Server.**

That's it — you're in.

---

## Path B — Modpack (optional, only if the server uses content mods)

If the server owner tells you there are extra mods installed, you'll need a modpack file (`.mrpack`) from them.

1. **Install [Prism Launcher](https://prismlauncher.org/)** (free, open-source).
2. Open Prism Launcher → click **Add Instance**.
3. Choose **Import** and drag-and-drop the `.mrpack` file the server owner sent you.
4. Click OK — Prism will download everything automatically.
5. Launch the instance and connect just like in Path A.

---

## Important: your username must match exactly

The server runs in offline mode, which means your in-game name is whatever you type in your launcher — **capitalization matters**.

> Example: `StevePlayer` and `steveplayer` are treated as two different people.

Before you try to join, tell the server owner the **exact spelling** of the username you use in your launcher. They need it to add you to the whitelist. If you get a "not whitelisted" error, double-check the spelling with them.
