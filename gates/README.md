# Verification Gates

In this repo, no "working/stable" claim is accepted without measurement.
Steps vary from environment to environment; these gates do not.
Whatever path you take, you prove success by passing all four gates.

---

| Gate | What it verifies |
|---|---|
| [saglik.md](saglik.md) | Server TPS/MSPT — is the game loop healthy |
| [otonomi.md](otonomi.md) | Zero-touch reboot + automatic resurrection after a crash |
| [erisim.md](erisim.md) | External access over a real MC protocol handshake |
| [modlar.md](modlar.md) | Integrity of all jars and no incompatibility/missing dependencies |
