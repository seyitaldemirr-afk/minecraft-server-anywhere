# Code of Conduct

## Our pledge

We want this project to be a friendly, harassment-free place for everyone, regardless of
experience level, background, or identity.

## Expected behavior

- Be respectful and patient. Many contributors are beginners self-hosting for the first time.
- Give and accept constructive feedback gracefully.
- Assume good faith.

## Unacceptable behavior

- Harassment, insults, or discriminatory remarks.
- Personal or political attacks.
- Publishing others' private information (including someone's home IP, tokens, or keys).

## Scope

This applies to all project spaces — issues, pull requests, discussions — and to public
spaces when representing the project.

## Enforcement

Report unacceptable behavior by opening an issue or contacting the maintainer. Maintainers
may remove, edit, or reject contributions that violate this code, and may block repeat
offenders.

This Code of Conduct is adapted in spirit from the
[Contributor Covenant](https://www.contributor-covenant.org).
