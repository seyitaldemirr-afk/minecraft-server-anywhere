# Reference Case — Galaxy S9 (SM-G960F · Android 10)

This is **exactly what we ran on our S9.** It is not a recipe to copy blindly —
it is a **worked example to learn from.** If your environment differs (phone model,
Android version, access method, tunnel), do not run these commands verbatim —
use the decision framework under `../method/` to adapt to your environment, then
prove you passed the universal gates under `../gates/`.

- `ortam.md` — the concrete environment in which this setup was done.
- `adimlar.md` — the real step-by-step chronology (with references to raw logs).
- `scripts/` — the scripts we actually ran.
- `config/` — the server.properties template we used.
- `kanit/` — raw outputs (`raw/`) + measurement dashboard.
